# TikTok画像編集GUI

Google Sheets / Google Drive をデータ管理に使い、ローカルでTikTok用画像（9:16）を一括生成するツールです。  
本文管理はスプレッドシート中心、見た目調整はGUI中心で運用します。

## 主な機能

- スプレッドシートの内容を読み込んで一括処理
- ストーリーIDの範囲指定でバッチ処理
- 右側1枚プレビューで見た目確認
- テーマ / テロップ種類 / フォント / 文字色 / フチ色 / フチ太さのGUI調整
- ダブルクリック起動（初回セットアップ自動）

## 必要環境

- Windows
- インターネット接続（初回セットアップ時）

## 最短起動（他環境でも同じ）

1. このリポジトリをダウンロードまたは `git clone`
2. `Start-TikTok-Image-Editor-GUI.bat` をダブルクリック
3. 自動でブラウザが開いたら `http://127.0.0.1:4173` を使用

初回は自動で以下を実行します。

- Node.js が未導入ならポータブル版をダウンロード
- `npm ci` 実行
- `app-data/local-config.json` の最小設定作成

## 停止

- `Stop-TikTok-Image-Editor-GUI.bat` をダブルクリック

## スプレッドシート入力（最小）

### シート: `ストーリー一覧`

- A列: ストーリーID
- B列: タイトル
- C列: 素材画像ID（Drive fileId）

### シート: `シーン一覧`

- A列: ストーリーID
- B列: シーン番号（1〜20）
- C列: 本文（入力）

## GUI運用

- 全件実行: `シート内容で一括処理開始`
- 範囲実行: `開始ID` / `終了ID` を入力して `範囲指定で処理開始`
- スタイル保存: GUIのスタイルを全体反映
- 右側画像更新: Drive画像を再取得して最新表示

## Google認証（必要な場合）

通常の閲覧中心処理はログインなしで動く構成ですが、環境や権限設定でログインが必要な場合があります。  
その場合はGUIの `Googleログイン` を使用してください。
